Política de Privacidade
ÁPEX Saúde — Consultoria APS · Última atualização: Janeiro de 2026
Resumo simples: A ÁPEX Saúde coleta apenas os dados necessários para o funcionamento da plataforma. Não vendemos, compartilhamos ou usamos seus dados para publicidade. Todos os dados ficam armazenados localmente no seu navegador e você pode excluí-los a qualquer momento.
01
Quem somos

A ÁPEX Saúde é uma plataforma de consultoria e monitoramento para equipes de Atenção Primária à Saúde (APS), desenvolvida para auxiliar gestores e profissionais de saúde no acompanhamento dos indicadores previstos na NT 6/2025-DEAPS/SAPS/MS.

Esta Política de Privacidade descreve como coletamos, usamos, armazenamos e protegemos as informações dos usuários da plataforma, em conformidade com a Lei Geral de Proteção de Dados (LGPD) — Lei nº 13.709/2018.

02
Dados que coletamos

Coletamos apenas os dados estritamente necessários para o funcionamento da plataforma:

  • Dados de cadastro: nome completo, endereço de e-mail, município e plano escolhido
  • Dados de uso: indicadores preenchidos nos painéis de monitoramento (dados institucionais, não pessoais de pacientes)
  • Dados de configuração: preferências de tema (claro/escuro) e configurações do sistema
  • Dados de acesso: data e hora de criação da conta

Não coletamos: dados de saúde de pacientes, CPF, RG, dados bancários ou qualquer informação sensível além das listadas acima.

03
Como os dados são armazenados

A plataforma ÁPEX Saúde utiliza o armazenamento local do navegador (localStorage) para guardar os dados de cadastro e uso. Isso significa que:

  • Os dados ficam armazenados no seu próprio dispositivo, não em servidores externos
  • Não temos acesso remoto aos dados preenchidos nos painéis
  • Ao limpar os dados do navegador, todos os seus dados da plataforma são apagados automaticamente
  • Os dados não são sincronizados entre dispositivos diferentes
04
Uso da Inteligência Artificial

A plataforma utiliza a API da Anthropic (Claude) para gerar estratégias personalizadas. Ao utilizar o recurso "Gerar Estratégia com IA":

  • Os dados dos indicadores preenchidos (percentuais e notas) são enviados de forma anônima para processamento
  • Não são enviados: nome do usuário, e-mail, município ou qualquer dado de identificação
  • Os dados enviados à IA são tratados conforme a Política de Privacidade da Anthropic
  • As respostas geradas pela IA ficam armazenadas apenas localmente no seu navegador
05
Finalidade do tratamento de dados

Tratamos seus dados exclusivamente para:

  • Permitir o acesso e uso da plataforma ÁPEX Saúde
  • Personalizar a experiência com base no perfil e equipe selecionados
  • Gerar relatórios e estratégias solicitadas pelo próprio usuário
  • Comunicar atualizações e comunicados relevantes da plataforma

Não utilizamos seus dados para: publicidade, venda a terceiros, perfilamento comportamental ou qualquer finalidade não listada acima.

06
Seus direitos (LGPD)

Como titular dos dados, você tem os seguintes direitos garantidos pela LGPD:

  • Acesso: saber quais dados temos sobre você
  • Correção: corrigir dados incompletos, inexatos ou desatualizados
  • Exclusão: solicitar a remoção dos seus dados da plataforma
  • Portabilidade: receber seus dados em formato estruturado
  • Revogação do consentimento: cancelar sua conta a qualquer momento
  • Oposição: opor-se ao tratamento de dados em determinadas situações

Para exercer qualquer um desses direitos, entre em contato pelo e-mail indicado abaixo. Respondemos em até 15 dias úteis.

07
Segurança dos dados

Adotamos as seguintes medidas para proteger seus dados:

  • Senhas armazenadas de forma segura no localStorage do dispositivo
  • Comunicação com a API de IA via HTTPS (protocolo seguro)
  • Proxy intermediário (Cloudflare Workers) para proteção das chaves de API
  • Nenhum dado sensível é transmitido em texto simples
08
Cookies e tecnologias similares

A plataforma utiliza localStorage (tecnologia similar a cookies) exclusivamente para:

  • Manter sua sessão ativa após o login
  • Salvar suas preferências de tema e configurações
  • Armazenar dados de onboarding e perfil

Não utilizamos cookies de rastreamento, analytics de terceiros ou pixels de publicidade.

09
Alterações nesta política

Esta política pode ser atualizada periodicamente. Em caso de alterações significativas, notificaremos os usuários através de comunicado na plataforma com pelo menos 15 dias de antecedência. A versão vigente estará sempre disponível nesta página com a data da última atualização.
